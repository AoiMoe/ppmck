ppmck release 9 ex3.5 by BKC


**** このファイルについて

　ppmck release9を改変したものです。

　以下の機能が利用可能です。

・2A03の矩形波、三角波部分にスラー（らしき）機能
・発音時のトリガー発生を若干抑える機能
・ポルタメントもどき
・ハードウェアエンベロープ機能（トラックABのみ利用可）


**** スラー概要

　ここで実装されている「スラー」とは、キーオフ、エフェクトテーブルの初期化をせずに、
次のノートに進む機能です。結果として、各種エフェクトが途切れることなく、
キーオフもされることなく、ただ発音周波数のみが変更されます。
副作用として、各テーブルが初期化されなかった場合に起こる不具合がある可能性もあります。



**** 使い方

    1. ppmck release 9をこのパッケージにあるファイルで置き換える
    2. mmlでスラー(&)を使って楽しむ。


 @v0 = { 15,14,13,10,6,5 }
 A o4 @v0 @2
 A c c&g g


とすると効果を確認できます。

また、

 A SM cdefg SMOF

とすると、SMからSMOFまでがスムースな発音になります。


    3.ポルタメントもどきを使う

  A c PS g PS c

などとすると、ピッチの上下を楽しむことができます。
ただし、PSコマンド後の音程は正確ではありませんので、
正確な音程を望むにはノートによる再発音が必要になります。


    4.ハードウェアエンベロープ

EHコマンド
指定の後、必ずduty設定を行うこと（有効にならない)
ボリューム設定でエンベロープのディケイレートを指定できる
EHx,y

x : 1でエンベロープを有効に
y : 0でエンベロープのループ

例:

 A EH1,1 @4 cdefgab>c
